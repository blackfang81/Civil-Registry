# سامانه ثبت احوال

یک سامانه ساده برای جستجو و استعلام مشخصات شهروندان، با احراز هویت کاربران، محدودیت مصرف (سهمیه) و پنل مدیریت جداگانه.

---

## امکانات

- ثبت‌نام و ورود کاربران (با `User` جنگو)
- جستجوی تک‌فیلدی (نام، نام خانوادگی، کد ملی، شماره تلفن)
- فیلتر بر اساس نام پدر و بازه تاریخ تولد
- نمایش زمان جستجو (ثانیه) و تعداد نتایج
- محدودیت مصرف روزانه و ماهانه برای هر کاربر
- پنل مدیریت جدا از Django Admin
- تنظیم سقف مصرف کلی و اختصاصی هر کاربر
- اسکریپت ایجاد بیش از ۱۰۰ هزار رکورد نمونه

---

## پیش‌نیازها

- Python 3.10+
- pip

---

## نصب و راه‌اندازی

### ۱. کلون و ورود به پروژه

```bash
cd Civil-Registry
```

### ۲. ساخت محیط مجازی

```bash
python -m venv .venv
```

**ویندوز:**

```bash
.venv\Scripts\activate
```

**لینوکس / مک:**

```bash
source .venv/bin/activate
```

### ۳. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### ۴. اعمال مایگریشن‌ها

```bash
python manage.py migrate
```

### ۵. ساخت ادمین

```bash
python manage.py createsuperuser
```

> کاربر ادمین به‌صورت خودکار `is_staff=True` دارد و به پنل مدیریت دسترسی می‌گیرد.

### ۶. ایجاد داده نمونه (اختیاری)

```bash
python manage.py seed_citizens --count 100000
```

### ۷. اجرای سرور

```bash
python manage.py runserver
```

سپس در مرورگر باز کنید: [http://127.0.0.1:8000](http://127.0.0.1:8000)

| `/accounts/register/` | ثبت‌نام |

## نکات

- فرمت شماره تلفن در دیتابیس: `+989121111111`
- هر جستجو یک واحد از سهمیه کاربر کم می‌کند
- برای جستجوی تلفن حداقل ۷ رقم لازم است (جلوگیری از جستجوی گسترده مثل `+98`)
- حداکثر ۱۰۰ نتیجه در هر جستجو نمایش داده می‌شود
- دیتابیس پیش‌فرض: SQLite (`db.sqlite3`)
